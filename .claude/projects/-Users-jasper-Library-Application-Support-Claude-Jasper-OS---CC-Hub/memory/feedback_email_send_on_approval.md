---
name: Send emails on approval
description: When Jasper approves a drafted email with "Yep", "Yes", "Send it" or similar, send it immediately — do not just create a draft.
type: feedback
---

When a drafted email is shown to Jasper and he approves it (e.g. "Yep", "Yes", "Send it", "Go ahead"), send the email immediately using the appropriate send tool. Do not create a draft and wait for a second confirmation.

**Why:** Jasper had to send an email himself because I created a draft instead of sending after he approved it.

**How to apply:** Draft shown + approval received = send. If unsure whether a send tool is available, load it via ToolSearch before showing the draft so you're ready to fire on approval.
