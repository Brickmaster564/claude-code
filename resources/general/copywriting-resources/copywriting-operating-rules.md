# Copywriting Operating Rules

## Default delivery standard
- All copy deliverables must be formal, properly punctuated, and internal-review ready.

## Readability standard
- Default target is approximately Grade 6 readability.
- Write with short sentences, simple words, and clean structure.
- Avoid jargon unless necessary for technical accuracy.
- Prioritise clarity over cleverness.

## Application scope
- Headlines
- Ad copy
- Captions
- Scripts
- Funnel sections
- Emails

## Override rule
- If Jasper explicitly requests a different tone or reading level for a specific asset, follow that request.
