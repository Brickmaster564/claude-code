# Nalu Resources

Add these files for Nalu (podcast agency) copy:

| File | What goes in it |
|------|----------------|
| `copywriting-bible.md` | Nalu brand voice, tone, positioning, proven angles |
| `icp.md` | Nalu's ideal clients — who they are, what they want from a podcast agency |
| `voc.md` | Voice of customer — real phrases from client calls, testimonials, objections |
| `offers.md` | Nalu's service offerings, packages, mechanisms, proof points |
| `objections.md` | Common objections from prospects and how to handle them |

For client-specific copy (writing for a Nalu client), create subfolders:
- `nalu/{client-name}/` with the same file structure
