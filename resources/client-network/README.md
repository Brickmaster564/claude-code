# Client Network Resources

Each vertical gets its own folder. Inside each folder, add these files:

| File | What goes in it |
|------|----------------|
| `copywriting-bible.md` | Brand voice, tone rules, proven angles, swipe file, do's and don'ts |
| `icp.md` | Ideal customer profile — demographics, psychographics, what they care about, where they hang out |
| `voc.md` | Voice of customer — real phrases from calls, reviews, forums, surveys. Pains, fears, desires, objections. |
| `offers.md` | Current offers — what you're selling, the mechanism, proof points, risk reversals, CTAs |
| `objections.md` | Top objections and how to handle them. Real objections from real prospects. |

## Verticals

- `life-insurance/`
- `tax-relief/`
- `home-security/`
- `senior-care/`

Add new vertical folders as needed. The copywriter skill auto-detects them.
